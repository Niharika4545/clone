# Social Media Website
This is a social media website where users can follow and unfollow others, view notifications, search for users, view profiles, see suggested users, post, like posts, comment on posts, log out, log in (including Google login), and create accounts using email and password or Google. The application is built with React, Firebase, UUID, and Chakra UI.

### Features
Follow and unfollow users
View notifications
Search for users
View other people's profiles
See suggested users
Create posts
Like posts
Comment on posts
Log out
Log in with email/password or Google
Create an account with email/password or Google
### Technologies Used
React
Firebase
UUID
Chakra UI
